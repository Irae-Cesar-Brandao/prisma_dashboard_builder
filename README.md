<div class="box">
<h2>1. Project</h2>
<p>This section defines the main information of the systematic review.</p>

<ul>
<li><b>Project Title:</b> Name of the systematic review.</li>
<li><b>Authors:</b> List of researchers involved in the study.</li>
<li><b>Research Period:</b> Defines the search time window (start and end dates).</li>
</ul>
</div>

<div class="box">
<h2>2. Databases</h2>
<p>This area is used to register where the literature searches were performed.</p>

<ul>
<li>Allows adding multiple databases (e.g., PubMed, Scopus, Web of Science).</li>
<li><b>+ Add Database:</b> Inserts new search sources.</li>
</ul>
</div>

<div class="box">
<h2>3. Project Management</h2>
<p>Controls project saving and data handling features:</p>

<ul>
<li><b>Save Project:</b> Stores data locally in the browser.</li>
<li><b>Load Project:</b> Retrieves a previously saved project.</li>
<li><b>Delete Project:</b> Removes stored project data.</li>
<li><b>Backup JSON:</b> Exports a full backup of the project.</li>
<li><b>Import JSON:</b> Loads a previously exported backup file.</li>
</ul>
</div>

<div class="box">
<h2>4. Initial Screening Removal</h2>
<p>Stage for cleaning and filtering data before screening.</p>

<ul>
<li><b>Duplicates:</b> Number of duplicate records removed.</li>
<li><b>Out of Time Range:</b> Studies outside the selected period.</li>
<li><b>Not Related:</b> Studies not relevant to the topic.</li>
</ul>
</div>

<div class="box">
<h2>5. Screening</h2>
<p>Stage where studies are evaluated based on title and abstract.</p>

<ul>
<li><b>Screening Excluded:</b> Number of articles removed during initial screening.</li>
</ul>
</div>

<div class="box">
<h2>6. Eligibility</h2>
<p>Stage where full-text articles are assessed for eligibility.</p>

<ul>
<li><b>Eligibility Excluded:</b> Number of studies excluded after full-text review.</li>
</ul>
</div>

<div class="box">
<h2>7. Output Generation</h2>

<ul>
<li><b>Generate Dashboard:</b> Creates the PRISMA flow diagram automatically.</li>
<li><b>Generate Report:</b> Produces a structured systematic review report.</li>
<li><b>Export Report:</b> Exports the report as a TXT file.</li>
<li><b>Export Dashboard PDF:</b> Generates a PDF version of the PRISMA diagram.</li>
</ul>
</div>

<div class="box">
<h2>8. PRISMA Result</h2>
<p>Displays the visual PRISMA flow diagram representing all stages of the systematic review process.</p>
</div>

<div class="box">
<h2>9. PRISMA Report</h2>
<p>Shows the automatically generated textual report based on the input data.</p>
</div>
